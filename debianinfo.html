HOW TO RUN APPLICATIONS AS ROOT
Sun 24 January 2015 1:26:56 PM

Debian 8.0 TESTING "jessie" update: 47k Jan 23 2015 11:50 PM
1. Open terminal emulator
2. type gksudo
3. a prompt should pop up
4. Type the APPLICATION NAME (not common name) into the box
5. Select the user you would like to run it as (default: root)
6. Hit run
7. The application should start with your desired priveleges
